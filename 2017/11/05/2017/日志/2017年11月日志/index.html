  • 成都过渡平台
      • 统计在过渡平台学习过的学员，sql语句如下（统计学时递减排列）：
        select sum(setting.period),s.id, s.username, s.name, s.sex, s.organization, s.register_time from t_course_student t, t_course_learning_setting setting, t_student s where t.course_id = setting.course_id and t.is_passed = 1 and t.student_id = s.id group by s.id order by sum(setting.period) desc
      • 统计所有学员的学时情况，包括你们没有学习过的学员：
        select a.p_sum,b.id,b.username,b.name,b.sex,b.organization,b.register_time from (select sum(setting.period) as p_sum ,s.id, s.username, s.name, s.sex, s.organization, s.register_time from t_course_student t, t_course_learning_setting setting, t_student s where t.course_id = setting.course_id and t.is_passed = 1 and t.student_id = s.id group by s.id) as a right join t_student b on a.id = b.id order by a.p_sum desc
